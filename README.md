# sql-data-warehouse-project
Data Warehouse Project

Overview

This project implements a modern data warehouse designed to support analytics, reporting, and business intelligence use cases. It integrates data from multiple sources, transforms it into a clean and consistent format, and stores it in a scalable analytical database.

Objectives
Centralize data from multiple sources
Enable fast analytical queries
Ensure data quality and consistency
Support reporting and dashboarding

ETL / ELT Process
Extract data from source systems
Load data into staging tables
Transform data into analytical models
Load into final warehouse tables

Data Quality
Schema validation
Null checks
Duplicate detection
Automated tests
